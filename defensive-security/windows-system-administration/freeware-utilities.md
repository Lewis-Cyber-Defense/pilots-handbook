# Hxstasy
David's personal collection of sysadmin tools and generally convenient software. Includes Sysinternals, all NirSoft tools, and a bunch of PortableApps.
More info: https://dj.je/pages/hxstasy
Download: https://dj.je/hxstasy
> (for v25.11) Download is approximately 2.5GB, unpacked is ~5GB
{.is-warning}


# PortableApps Platform
https://portableapps.com/download

**Easy grabs:**
- Firefox Portable
- Everything
- 7zip
- Detect It Easy
- Process Hacker

Also has Sysinternals tools, but I've had mixed success getting those to work when downloaded from PortableApps.
