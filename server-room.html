Remote access

The infrastructure is configured with Tailscale cleints for remote access. To access the tailnet, you must be a member of the Lewis-Cyber-Defense Github organization. Once you are a member, install and log in to Tailscale using your Github account, and select the lewis-cyber-defense.org.github network. Ask any infrastructure admin for personal credentials to log in to the infrastructure.

At the time of writing, the following systems can be accessed directly: app1, app2, pfsense
In order to reach other systems, you will have to jump via one of the systems above.
